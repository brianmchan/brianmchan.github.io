Math 480B: April 13, 2014

I am finally writing my first blog post. This quarter is my last as an undergraduate at the University of Washington and I want to finish my undergraduate career on a good note.

I only need one more math class to fulfill my degree and I found the perfect class that was being offered in Spring 2014. I also wanted to learn more about Python. I am currently taking Math 480B, which is using Sage, an open source mathematical software using Python. Many people say it's a great class and I hope to learn a lot from the class. I am learning a lot in that class. List comphrension is pretty cool and it's filling in the gaps about Python that I haven't learned before.

Example of list comphrension:

my_data = [n**Integer(2) for n in range(Integer(10))]

The cool thing about list comprehensions is that it is concise and easy to understand and instead of writing out a loop, everything can be written in just one line.

As for Sage, I am liking it so far. It is much better than using MATLAB. Also, Professor Stein creating the SageMathCloud website is a great idea. Sure, it has some bugs (like refreshing the browser causes it to go back to the login screen), but not needing to install anything and programming on a web browser is great.
